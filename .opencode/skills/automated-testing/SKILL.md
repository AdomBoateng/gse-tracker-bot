# Automated Testing

## Purpose
Enable agents to reduce manual testing effort and detect issues early.

## When to Use This Skill
Use this skill when setting up or expanding a test automation suite, configuring CI test runs, or reducing manual QA effort.


## Rules
1. Ensure tests are reliable and repeatable.
2. Avoid flaky tests that produce inconsistent results.
3. Keep test scripts modular and maintainable.

## Output Format
Provide:
- Test suite structure and file locations
- Commands to run tests
- Summary of coverage and pass/fail results
- List of any flaky or skipped tests with reasons
