# Backend Optimization

## Purpose
Ensure backend services handle high traffic and resource-intensive workloads efficiently.

## When to Use This Skill
Use this skill when backend services experience high latency, excessive resource usage, or scalability bottlenecks.


## Rules
1. Avoid single points of failure.
2. Always test under realistic load scenarios.
3. Document all optimizations and configurations.

## Output Format
Provide:
- Current bottleneck analysis
- Proposed optimizations with expected impact
- Before/after benchmark comparisons where possible
- Implementation steps and rollback plan
