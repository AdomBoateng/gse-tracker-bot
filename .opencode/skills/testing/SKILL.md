# Testing

## Purpose
Enable agents to validate functionality, performance, and reliability of systems.

## When to Use This Skill
Use this skill when writing test cases, running regression tests, or validating system behavior before a release.


## Rules
1. Maintain high test coverage.
2. Always validate critical paths first.
3. Document test cases and outcomes.

## Output Format
Provide:
- Test plan with scope and methodology
- Test case list with expected outcomes
- Coverage report (lines, branches, critical paths)
- Summary of passed, failed, and skipped tests
