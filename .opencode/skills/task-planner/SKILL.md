# Task Planner

## Purpose
Enable agents to create actionable, prioritized task lists that maximize efficiency.

## When to Use This Skill
Use this skill when a high-level goal needs to be decomposed into specific, assignable, and trackable subtasks.


## Rules
1. Ensure tasks are specific, measurable, and trackable.
2. Avoid conflicting task assignments.
3. Reassess priorities periodically based on progress.

## Output Format
Provide:
- Ordered task list with descriptions and effort estimates
- Dependency map between tasks
- Assigned agent or role per task
- Priority tiers (critical / high / medium / low)
