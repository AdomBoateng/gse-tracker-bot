# Prompt Optimization

## Purpose
Enable agents to improve prompt clarity, relevance, and cost-efficiency for LLMs.

## When to Use This Skill
Use this skill when an existing prompt is too verbose, produces inconsistent outputs, or exceeds token budget.


## Rules
1. Always validate optimized prompts with diverse queries.
2. Keep prompt history for reproducibility.
3. Avoid ambiguous or overly broad instructions.

## Output Format
Provide:
- Original vs. optimized prompt comparison
- Token count reduction and quality impact metrics
- Test results across diverse queries
- Recommended variant with justification
