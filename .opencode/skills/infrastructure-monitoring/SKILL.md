# Infrastructure Monitoring

## Purpose
Enable agents to detect and respond to infrastructure issues proactively.

## When to Use This Skill
Use this skill when setting up observability, configuring alerts, or investigating infrastructure performance issues.


## Rules
1. Ensure monitoring does not introduce significant overhead.
2. Prioritize high-impact alerts.
3. Maintain historical performance data for analysis.

## Output Format
Provide:
- Monitoring configuration (dashboards, alert rules)
- List of tracked metrics with thresholds
- On-call escalation procedure
- Historical baseline summary if available
