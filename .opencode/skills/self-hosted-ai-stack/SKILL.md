# Self-Hosted AI Stack

## Purpose
Enable agents to deploy, monitor, and maintain AI models in a self-hosted environment.

## When to Use This Skill
Use this skill when deploying or maintaining local LLMs, vector databases, or AI inference infrastructure.


## Rules
1. Ensure high availability and fault tolerance.
2. Prioritize open-source and internally hosted tools.
3. Document all configurations and procedures.

## Output Format
Provide:
- Stack architecture (models, vector DBs, APIs, orchestration)
- Configuration files and startup commands
- Health check and monitoring instructions
- Backup and recovery procedures
