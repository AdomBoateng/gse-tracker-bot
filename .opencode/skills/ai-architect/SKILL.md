# AI Architect

## Purpose
Enable agents to design reliable, maintainable, and scalable AI architectures for production.

## When to Use This Skill
Use this skill when designing a new AI system, selecting models or infrastructure, or planning integrations between AI components.


## Rules
1. Design for modularity and maintainability.
2. Avoid single points of failure in the architecture.
3. Prioritize open-source tools when possible.

## Output Format
When producing an architecture, provide:
- Component list with roles and responsibilities
- Data flow description between components
- Technology choices with justification
- Scalability and fault-tolerance notes
- Known trade-offs or risks
