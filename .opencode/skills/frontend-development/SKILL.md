# Frontend Development

## Purpose
Enable agents to develop high-quality web and mobile interfaces that are visually appealing, user-friendly, and performant.

## When to Use This Skill
Use this skill when building or updating UI components, pages, or full frontend applications.


## Rules
1. Prioritize clean, maintainable, and modular code.
2. Use automated testing for UI components.
3. Follow company branding and UX guidelines.
4. Avoid hardcoding data; always integrate with backend or mock APIs for development.

## Output Format
Provide:
- Component structure and file layout
- Key implementation decisions (framework, state management)
- Instructions to run and test the frontend
- Accessibility and responsiveness notes
