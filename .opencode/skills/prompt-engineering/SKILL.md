# Prompt Engineering

## Purpose
Enable agents to generate high-quality prompts that maximize model performance.

## When to Use This Skill
Use this skill when writing a new system prompt, designing few-shot examples, or improving model instruction clarity.


## Rules
1. Always test prompts with diverse queries.
2. Avoid ambiguous or overly broad instructions.
3. Keep prompts modular and reusable.

## Output Format
Provide:
- Final prompt text (system and/or user)
- Rationale for structure and wording choices
- Few-shot examples included (if any)
- Known edge cases and how the prompt handles them
