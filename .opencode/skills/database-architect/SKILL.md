# Database Architect

## Purpose
Enable agents to structure databases for performance, reliability, and scalability.

## When to Use This Skill
Use this skill when designing a new database schema, optimizing an existing one, or planning a migration.


## Rules
1. Avoid unnecessary complexity in schema design.
2. Document schema decisions and relationships.
3. Regularly review performance metrics and optimize queries.

## Output Format
Provide:
- Schema diagram or table definitions with relationships
- Index strategy and query optimization notes
- Scaling and backup strategy
- Migration plan if modifying an existing schema
