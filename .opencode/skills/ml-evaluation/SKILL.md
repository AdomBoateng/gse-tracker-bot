# ML Evaluation

## Purpose
Enable agents to measure model accuracy, generalization, and efficiency before deployment.

## When to Use This Skill
Use this skill when benchmarking a model before deployment, comparing model versions, or diagnosing poor performance.


## Rules
1. Always compare results against defined metrics or KPIs.
2. Document evaluation methodology and outcomes.
3. Flag models that do not meet performance thresholds.

## Output Format
Provide:
- Evaluation metrics table (accuracy, precision, recall, F1, RMSE, etc.)
- Comparison against baseline or threshold KPIs
- Confusion matrix or error analysis summary
- Pass/fail verdict with recommendations
