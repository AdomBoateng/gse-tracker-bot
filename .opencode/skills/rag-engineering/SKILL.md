# RAG Engineering

## Purpose
Enable agents to build reliable Retrieval-Augmented Generation pipelines.

## When to Use This Skill
Use this skill when building a new RAG pipeline, integrating a vector store with an LLM, or designing document retrieval flows.


## Rules
1. Validate retrieval results for correctness.
2. Use memory and embeddings efficiently.
3. Ensure RAG pipeline scales with data size.

## Output Format
Provide:
- Pipeline architecture (ingestion → indexing → retrieval → generation)
- Vector database and embedding model choices with rationale
- Sample query and retrieved context example
- Latency and accuracy benchmarks
