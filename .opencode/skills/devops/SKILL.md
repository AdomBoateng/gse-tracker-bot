# DevOps

## Purpose
Enable agents to deploy, monitor, and maintain production systems efficiently.

## When to Use This Skill
Use this skill when deploying services, managing infrastructure, setting up CI/CD, or handling production incidents.


## Rules
1. Maintain configuration as code.
2. Ensure secure and scalable deployments.
3. Use monitoring alerts to detect issues early.

## Output Format
Provide:
- Infrastructure-as-code files (Dockerfile, Helm chart, Terraform, etc.)
- Deployment steps and rollback procedure
- Monitoring and alerting configuration
- Environment-specific variable requirements
