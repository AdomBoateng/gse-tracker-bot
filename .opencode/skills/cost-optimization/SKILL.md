# Cost Optimization

## Purpose
Enable agents to minimize resource usage without compromising quality.

## When to Use This Skill
Use this skill when reducing infrastructure, API, or compute costs, or when analyzing resource usage for efficiency.


## Rules
1. Ensure cost reductions do not degrade critical functionality.
2. Document optimizations for transparency.
3. Continuously reassess for new savings opportunities.

## Output Format
Provide:
- Current cost breakdown by resource or API call
- Specific optimization recommendations with estimated savings
- Trade-off analysis for each recommendation
- Implementation priority order
