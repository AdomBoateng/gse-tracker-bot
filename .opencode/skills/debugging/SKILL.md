# Debugging

## Purpose
Enable agents to troubleshoot issues efficiently and restore system stability.

## When to Use This Skill
Use this skill when a runtime error, unexpected behavior, or performance issue needs root-cause analysis and a fix.


## Rules
1. Document root cause analysis.
2. Prioritize critical and high-impact bugs.
3. Avoid introducing new issues when fixing errors.

## Output Format
Provide:
- Root cause analysis with evidence (logs, traces)
- Steps to reproduce the issue
- Proposed fix with code changes
- Regression test to prevent recurrence
