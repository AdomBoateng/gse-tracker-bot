# Backend Development

## Purpose
Enable agents to design and implement backend systems that are secure, performant, and able to scale with traffic and user growth.

## When to Use This Skill
Use this skill when building new APIs, services, or backend modules, or when refactoring existing backend code.


## Rules
1. Write modular, maintainable, and well-documented code.
2. Use unit and integration tests for reliability.
3. Ensure APIs are backward-compatible when making updates.
4. Always consider horizontal and vertical scaling.

## Output Format
Provide:
- API design (endpoints, methods, payloads)
- Module/service structure
- Key implementation decisions with rationale
- Instructions to run or test the service
