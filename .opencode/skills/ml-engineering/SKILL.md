# ML Engineering

## Purpose
Enable agents to build production-ready ML pipelines with high reliability.

## When to Use This Skill
Use this skill when building, training, or deploying machine learning models or end-to-end ML pipelines.


## Rules
1. Ensure models are explainable and auditable.
2. Monitor for performance drift and retrain when necessary.
3. Prioritize efficiency and scalability.

## Output Format
Provide:
- Dataset summary and preprocessing steps
- Model architecture and training configuration
- Evaluation metrics and benchmark results
- Serving instructions and versioning notes
