# Evaluator

## Purpose
Enable agents to assess the accuracy, completeness, and performance of solutions.

## When to Use This Skill
Use this skill when assessing the quality, correctness, or completeness of an agent output, model result, or system behavior.


## Rules
1. Maintain objectivity in evaluation.
2. Use metrics and benchmarks where possible.
3. Document evaluation results for auditing.

## Output Format
Provide:
- Evaluation scores per criterion (accuracy, completeness, performance)
- Comparison against baseline or expected results
- Identified gaps with improvement suggestions
- Overall pass/fail verdict with confidence level
