# Data Engineering

## Purpose
Enable agents to handle structured and unstructured data efficiently.

## When to Use This Skill
Use this skill when building or maintaining ETL pipelines, data ingestion flows, or data storage systems.


## Rules
1. Maintain consistency and accuracy of data.
2. Use scalable open-source data storage and processing tools.
3. Ensure pipelines are modular and maintainable.

## Output Format
Provide:
- Pipeline architecture diagram or description
- Source-to-destination data flow
- Transformation logic and validation checkpoints
- Deployment and scheduling instructions
